# Product

## Register

brand

## Users

People who need a ready-to-use PDF form, template, or invoice, plus developers receiving the implementation as a design handoff.

## Product Purpose

Help visitors quickly find an appropriate document, understand its purpose from the preview and copy, and start using it without unnecessary catalog friction. The repository provides a faithful, maintainable implementation of the approved TheBestPDF library designs.

## Brand Personality

Practical, confident, and clear. The interface should feel professional and helpful without sounding legalistic or decorative.

## Anti-references

- Generic placeholder or AI-generated document previews.
- Visual changes that drift from the approved Figma layouts, Inter typography, established radii, or TheBestPDF colors.
- Catalog controls that remain visible when they cannot help the user complete the current task.

## Design Principles

- Make the document preview and title the primary decision inputs.
- Keep Templates, Forms, and Invoice libraries structurally consistent.
- Show controls only when they are relevant to the current state.
- Preserve approved source content and assets instead of inventing substitutes.
- Keep desktop and mobile behavior explicit and independently testable.

## Accessibility & Inclusion

Maintain keyboard-visible focus, semantic controls and status messages, readable contrast, descriptive image alternatives, responsive layouts without horizontal overflow, and reduced-motion behavior.
