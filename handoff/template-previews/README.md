# Template preview library

This folder is the handoff copy of every preview used by the templates catalog. Folder names are category labels; file names are the user-facing template titles.

| Folder | Category | Files |
| --- | --- | ---: |
| `receipts/` | Receipts | 8 |
| `estimates/` | Estimates | 6 |
| `business-agreements/` | Business Agreements | 11 |
| `financial-statements/` | Financial Statements | 2 |
| `business-letters/` | Business Letters | 5 |
| `contracts/` | Contracts | 17 |
| `bill-of-sale/` | Bill of Sale | 7 |
| **Total** | **7 categories** | **56** |

Use [`manifest.csv`](manifest.csv) to map each preview to its category, title, description, and handoff path. Runtime code continues to use `public/assets/template-previews/`; this folder is intentionally organized for developer delivery.
